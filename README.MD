# Portfolio Template

A clean, modular, and reusable personal portfolio template built using HTML, CSS, and vanilla JavaScript.

---

## Structure

- index.html → Home
- about.html → About section
- projects.html → Work showcase
- contact.html → Contact page
- css/style.css → Styling
- js/main.js → Interactions
- assets/ → Images and media

---

## Purpose

This is a general-purpose portfolio template designed to be:
- Lightweight
- Easy to customize
- Expandable for future projects

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
